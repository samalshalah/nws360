Transform NWS360 into a commercial SaaS platform with clear plans, seat-based users, onboarding, and client-facing experience. The goal is to make the system easy to purchase, activate, and scale inside organizations.

1) Subscription Plans With Seats (Core Business Model)

Implement subscription tiers:

Basic
Pro
Enterprise


Each plan controls both features and number of users (seats).

Plan	Users	Keywords	Sources	Analytics	AI Brief	API
Basic	3 users	10	limited	standard	summary	no
Pro	10 users	50	extended	advanced	full	limited
Enterprise	unlimited	unlimited	unlimited	full	custom	full

Configuration:

plan_limits = {
  basic: { users: 3, keywords: 10, sources: limited },
  pro: { users: 10, keywords: 50, sources: extended },
  enterprise: { users: -1, keywords: -1, sources: unlimited }
}


-1 means unlimited.

Store plan in:

subscriptions(client_id, plan, max_users, renewal_date, status)

2) Seat Enforcement (User Limits)

Each user must belong to a client:

users.client_id


When creating a user:

IF active_users >= plan.max_users
  → block creation
  → show upgrade message


Admin panel must display:

Team Usage: 7 / 10 seats used


Admin actions:

invite user

deactivate user (frees seat)

change role

resend invite

Roles per seat:

Admin | Analyst | Editor | Viewer
